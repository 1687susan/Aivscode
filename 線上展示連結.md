## 📱 手機版快速連結
建議使用方案一的 htmlpreview 連結，在手機上也能正常顯示。

## 🚀 GitHub Pages 啟用步驟（供參考）
1. 進入 GitHub 專案頁面
2. 點擊 Settings
3. 找到 Pages 設定
4. Source 選擇 "Deploy from a branch"
5. Branch 選擇 "main"
6. 點擊 Save

啟用後網址將是：https://1687susan.github.io/Aivscode/
